先确认准备项

01

安装 Codex 前，不需要你已经会写代码，但电脑环境要先满足几个基础条件。

✓
一台电脑

推荐 macOS 或 Linux。Windows 用户建议使用 WSL。

✓
Node.js 和 npm

Codex CLI 需要通过 npm 安装，后面会教你检查。

✓
稳定网络

需要能正常访问 ChatGPT / OpenAI 相关服务。

✓
一个项目文件夹

可以是真项目，也可以先新建一个空文件夹练手。

网络打不开怎么办？

如果你还不知道如何科学上网，或者访问 ChatGPT 时经常打不开、加载失败、登录不了，可以私信我，我单独发你一份小白可操作的解决方案。

正式安装 Codex

02

打开终端

1

macOS 用户按下 Command + 空格，搜索“终端”或 Terminal，然后回车打开。

检查 npm

2

输入下面的命令。如果出现版本号，说明 npm 已经装好。

npm -v

安装 Codex

3

复制下面的命令到终端，等待安装完成。

npm install -g @openai/codex

确认安装成功

4

如果能看到版本号，说明 Codex 已经安装成功。

codex --version

登录 Codex

5

运行登录命令后，浏览器会打开授权页面。按提示登录你的 ChatGPT 账号即可。

codex --login

进入项目文件夹

6

如果你还没有项目，可以先新建一个练手文件夹。

mkdir my-first-codex-project
cd my-first-codex-project

启动 Codex

7

在项目文件夹里输入 codex，就可以开始用中文提需求。

codex

新手先这样用

03

刚开始不要急着让 Codex 全自动干活。先让它解释、规划、再修改，你会更容易掌控结果。

默认模式 适合新手。Codex 会先提出建议，你确认后再执行。
自动编辑模式 适合重复修改。Codex 可以自动改文件，但运行命令前仍会询问。
全自动模式 适合熟悉后使用。适合修构建、跑测试、原型开发等长任务。
小白原则 不懂就先问解释，重要修改前先让它说计划。

第一次可以这样问

04
请解释这个项目的目录结构，用小白能听懂的话说。
请帮我找出这个项目怎么启动。
请帮我创建一个简单的个人介绍网页，包含头像、名字、简介和联系方式。
请先不要改代码，只告诉我你的修改计划。
每次修改后，请告诉我改了哪些文件，以及我应该怎么验证。

常见问题

05
我不会写代码，可以用 Codex 吗？

可以。你负责说需求，Codex 负责解释、修改和执行。刚开始建议让它一步步说明。

安装失败怎么办？

先检查 Node.js 是否安装、npm 是否可用、网络是否稳定。也可以把终端里的报错复制给 Codex，让它帮你判断原因。

Codex 和 ChatGPT 有什么区别？

ChatGPT 更像对话助手；Codex 更像可以进入项目文件夹工作的编程助手，能读文件、改文件、运行命令。

新手最该记住哪几个命令？

先记住 npm -v、npm install -g @openai/codex、codex --login、cd 项目文件夹、codex。

最后：账号和会员

06

你的 ChatGPT 账号需要先完成登录。想更稳定地使用 Codex，建议升级到 Plus 或 Pro 会员后再使用。

如果你只是第一次学习和跑通流程，先选 Plus 通常就够了。如果你每天都要长时间使用 Codex 做项目，再考虑 Pro。

Plus 会员 适合大多数新手和日常使用：学习、做网页、写脚本、改项目。
Pro 会员 适合高频开发用户：更长时间、更复杂任务、更多 Codex 使用需求。

升级路径：打开 chatgpt.com，进入账号菜单，点击 Settings / 设置，找到 My Plan / 我的方案，然后选择 Upgrade / 升级。也可以打开 OpenAI 官方价格页查看当前方案。

https://openai.com/chatgpt/pricing/
